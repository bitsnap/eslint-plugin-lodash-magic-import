Performs simple lodash usage checks, nothing fancy.

## How to use 

```
> npm i --save-dev eslint-plugin-lodash-magic-import babel-eslint eslint lodash
```

*.eslintrc*
```
{
  "parser": "babel-eslint"
  "plugins": ["lodash-magic-import"],
  "extends": ["plugin:lodash-magic-import/recommended"]
}
```
